# Customer

The `Customer` model defines the key information needed to identify and communicate with a delivery recipient.

## Fields

### `id`
- **Type:** `number`
- **Attributes:** Read-only, Unique  
  A unique identifier assigned to each customer.

### `name`
- **Type:** `string`
- **Attributes:** Required  
  The full name of the customer.

### `phone_number`
- **Type:** `string`
- **Attributes:** Required  
  The customer's primary phone number, including the country code.

### `secondary_phone_number`
- **Type:** `string`
- **Attributes:** Optional  
  An alternative phone number for the customer, also including the country code.

### `email`
- **Type:** `string`
- **Attributes:** Optional  
  The customer's email address.
